# Ametscript
A statically typed object oriented programming language written in python (restarting)

(still being worked on).

FEATURES:
- [ ] Variable Classes
- [ ] interpreter
- [ ] functions
- [ ] classes 
- [ ] python api
- [ ] modules
- [ ] standard library
- [ ] basic syntax
